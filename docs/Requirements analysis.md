## 1 功能声明

### 1.1 销售与工作数据

+ 采购信息：某段时间内，按照采购顺序，图书的 ISBN 号、数量、单价等。
+ 销售情况：某段时间内，按照销售顺序，图书的 ISBN 号、数量、单价等。
+ 盈利信息：某段时间内，收入、支出、利润。
+ 工作报告：某段时间内，按照操作顺序，每次操作类型及其对象。
+ 工作日志：系统工作信息。

### 1.2 书库

+ 每种书包含信息：
  + ISBN 号：每本 20 字符。
  + 书名：60 字符。
  + 作者：60 字符。
  + 关键字：多个，总共不超过 60 字符。
  + 库存量：整型。
  + 销售单价：非负实数。
+ 操作：
  + 录入：输入一种书的完整信息，在第一次采购本种书时进行。
  + 采购：ISBN 号，数量，进货价格。
  + 查询：根据 ISBN 号、作者名字、关键字、书名中的任意组合进行查询。
  + 购买：通过给定的 ISBN 号和购买数量来进行操作。

### 1.3 用户

+ 游客：
  + 包含信息：
    + 用户名。
    + 密码。
  + 权限：
    + 注册、登陆。
+ 顾客（在游客的基础上）：
  + 包含信息：
    + 操作记录（两种，见下）。
  + 权限：
    + 查询书籍。
    + 购买书籍。
+ 销售人员（在顾客的基础上）：
  + 包含信息：
    + 操作记录（四种，见下）。
  + 权限：
    + 录入书籍。
    + 采购书籍。
    + 书籍信息修改。
    + 用户创建。
+ 店长（在销售人员的基础上）：
  + 权限：
    + 查询采购信息。
    + 查询销售情况。
    + 查询盈利信息。
    + 查询工作报告。
    + 查询工作日志。
  + 额外：在系统首次运行时注册。
